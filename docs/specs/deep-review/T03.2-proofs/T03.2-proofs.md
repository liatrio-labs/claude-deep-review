### Key Logic Details

- `SEVERITY_ORDER = ["critical", "high", "medium", "low"]` — lower index = higher severity
- Security dimensions use `min(confidence_threshold, security_min_confidence)` as effective threshold
- Severity comparison uses index: `sev_idx > sev_threshold_idx` means "below threshold"
- Each eliminated finding gets `eliminated_by = "threshold"` and `elimination_reason`
- Eliminated findings are dict copies — originals are not mutated
